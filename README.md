# ASPP

### ASPP - Автоматический запуск проекта Python



Этот проект был создан для автоматизации запуска проектов на YAP Python на ПК. Проект также подразумевает автоматический перезапуск проекта при его выходе из строя.



### руководство пользователя



1) Выберите предпочитаемую вами операционную систему;

2) Выберите и запустите .py файл (или exe-файл для ОС Windows);

3) Введите необходимые данные;

*) Программа скомпилирует исполняемый файл для требуемой ОС и добавит его в автозапуск системы, этот файл будет запускать проект по введенному вами пути при каждом запуске ОС (и перезапускать по мере необходимости). Если проекта нет, исполняемый файл завершит свое выполнение.


Наслаждайтесь его использованием :)

### ASPP - Auto Starting Python Project

This project was created to automate the launch of projects on YAP Python on a PC. The project also implies an automatic restart of the project when it breaks down.

### manual

1) Choose your preferred OS;
2) Select and run the .py file (or .exe file for OS windows);
3) Enter the required data;
*) The program will compile an executable file for the required OS and add it to the autorun of the system, this file will run the project on the path you entered each time the OS is started (and restart as needed). If there is no project, the executable file will complete its execution.

Enjoy using it :)
